# Rencana Integrasi MT5 ke Hunter Trades Journal

## Latar Belakang
- **Status saat ini**: User/member input manual log trading via form
- **Tujuan**: Connect MT5 agar posisi entry trading otomatis tersync ke journal
- **Preferensi_USER**:
  - Platform: MT5 (MetaTrader 5)
  - Konflik data: Prioritas MT5
  - Posisi: Semua posisi

## Arsitektur yang Dibutuhkan

### 1. Update Schema Database

Di `server/src/models/TradingAccount.ts`:
```typescript
// Tambahan field untuk MT5
mt5Config: {
  server: string;      //Contoh: "MetaQuotes-Demo"
  login: string;       //Account MT5 number
  password: string;   //Master password MT5
}
sourcePreference: "manual" | "mt5"; //Default: manual
lastMt5SyncAt?: Date;
```

Di `server/src/models/Trade.ts`:
```typescript
// Tambahan field untuk tracking source
source: "manual" | "mt5";
mt5TicketId?: string;  //Ticket/Order ID dari MT5
mt5OrderId?: string;
```

### 2. Service Layer Baru

Buat file baru: `server/src/services/mt5.service.ts`

Fungsi:
- `connect(config)`: Connect ke MT5 WebAPI
- `getOpenPositions()`: Ambil semua posisi terbuka
- `getClosedPositions(from, to)`: Ambil history trading
- `syncPositions()`: Sync posisi dari MT5 ke journal
- `disconnect()`: Cleanup connection

### 3. API Endpoints Baru

Di `server/src/routes/mt5.routes.ts`:

| Method | Endpoint | Deskripsi |
|-------|----------|----------|
| POST | /api/mt5/connect | Connect ke MT5 account |
| POST | /api/mt5/disconnect | Disconnect |
| GET | /api/mt5/positions | Get posisi terbuka di MT5 |
| POST | /api/mt5/sync | Manual sync trigger |
| GET | /api/mt5/status | Get connection status |

### 4. Cron Job untuk Auto Sync

Buat: `server/src/services/mt5-scheduler.service.ts`

- Interval sync:Setiap 1-5 menit (bisa dikonfigurasi)
- Sync hanya posisi yang belum tersync
- Logika konflik:Jika ada trade manual dengan ticket sama, overwrite dengan data MT5

### 5. Frontend Pages

**halaman baruyang perlu dibuat/diupdate**:

1. `Settings → MT5 Connection`
   - Form input MT5 server, login, password
   - Toggle auto-sync
   - Test connection button
   - Last sync status

2. `Dashboard → Source Toggle`
   - Pilihan: Manual Entry / MT5 Sync
   - Indicator status koneksi MT5

## Catatan Teknis

### MT5 WebAPI
MT5 menyediakan WebAPI resmi untuk:
- Get positions
- Get orders history
- Get account info

Official documentation: https://www.mql5.com/en/docs/http

### Alternative: MQTT/WebSocket
Jika MT5 WebAPI tidak cukup, bisa gunakan:
- EA (Expert Advisor) di MT5 yang publish ke webhook
- WebSocket connection untuk real-time update

### Keamanan
- MT5 password dienkripsi sebelum disimpan
- Gunakan environment variables untuk credential
- Rate limiting untuk MT5 API calls
